1. 리스트 (list)
    1) 많은 양의 자료를 사용해야 할 경우
    2) 여러 개의 데이터를 의미 있게 묶어서 저장
    3) 순서 O, 중복 O, 수정 O, 삭제 O
    4) 인덱스(index) : 리스트의 항목의 위치를 알려주는 번호
        - 리스트에서 인덱스는 0부터 시작
    5) 슬라이싱(slicing)    
        - 여러 개의 항목을 추출하는 기법

2. 튜플 (tuple)
    1) 리스트와 유사
    2) 리스트는 대괄호 []를 이용하지만, 튜플은 소괄호 ()를 사용함
    3) 순서 O, 중복 O, 수정 X, 삭제 X
    4) 값이 변경되면 안되는 경우 튜플을 사용하면 효과적임

3. 사전(Dictionary) 자료형
    1) 특정한 원소의 존재 여부를 빠르게 확인하고 추가/삭제 가능
    2) 데이터를 키(key)와 값(value) 쌍의 형태로 저장할 때 사용
    3) 사전 데이터[키] = 값
    4) 순서 X, 중복 X, 수정 O, 삭제 O

4. 집합(Set)
    1) 순서 X, 중복 X
