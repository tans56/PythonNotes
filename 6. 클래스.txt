1. 클래스 정의
    1) 클래스 변수
        - 직접 사용 가능, 객체보다 먼저 생성 
    2) 인스턴스 변수
        - 객체마다 별도로 존재, 인스턴스 생성 후 사용 
        - 예) self.name : 현재 인스턴스의 name 변수
              speed : 함수의 파라미터로 넘어 온 speed 변수
        - 해당 인스턴스가 가지는 값과 함수의 인자 값을 구분함.  
        - 각 인스턴스의 인스턴스 변수는 서로 다름
        - 구체적인 하나의 인스턴스에서 사용하는 변수        
    3) self
        - 인스턴스(instance) 자기 자신을 의미함
    4) 생성자
        - 인스턴스가 생성될 때 자동으로 실행
        - init() 메서드        
2. 객체 생성
    1) 네임스페이스
        - 객체를 인스턴스화 할 때 저장된 공간

3. 정보 은닉
    1) 인스턴스 변수를 private으로 정의하려면
       변수 이름 앞에 __을 붙이면 됨
    2) private이 붙은 인스턴스 변수는 클래스 내부에서만 접근 가능함
    3) getter
        - 인스턴스 변수값 반환
    4) setter
        - 인스턴스 변수값 설정
