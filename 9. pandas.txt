1. 판다스 개념
    1) 파이썬의 데이터 분석 라이브러리
    2) 기본적으로 넘파이를 사용
        - 파이썬에서 배열을 다루는 최적의 라이브러리
        - 인덱싱, 연산, 전처리 등 다양한 함수 제공
    3) 데이터프레임(DataFrame) : 데이터 테이블 전체 객체   
       시리즈(Series) 
        - 각 열을 데이터를 다루는 객체
        - 넘파이 배열(ndarray)의 하위 클래스
        - 리스트, 튜플, 넘파이 배열(1차원) - 한 열 혹은 한 행
        - 세가지 요소(property)
            - data : 값을 저장하는 요소
            - index : 항상 0부터 시작 숫자, 문자열,
                      순서가 일정하지 않은 숫자를 입력가능
                      인덱스 중복 허용
            - dtype : 넘파이의 데이터 타입과 일치
